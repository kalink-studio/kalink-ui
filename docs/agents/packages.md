# Packages

## Canopy

- Hosts Payload CMS elements such as custom fields and custom blocks.
- When you add a new custom field or block to a Payload project and the intent is not obvious, ask whether it should be added to the canopy package.

## Seedly

- Hosts design-system components and styling primitives in `packages/seedly/src/components`.
- Before creating or refactoring Seedly components, read `docs/agents/seedly-components.md`.
- Reuse existing shared component styles when parity is intended instead of duplicating parallel contracts.
